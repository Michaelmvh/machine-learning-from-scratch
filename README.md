## Running local environment

1. Activate environment

   `source machine-learning-from-scratch/bin/activate`

2. Install package

   `pip install <package-name>`

3. Update requirements.txt

   `pip freeze > requirements.txt`

4. Deactivate environment

   `deactivate`
